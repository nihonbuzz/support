# Nihonbuzz ロゴ

ロゴは Nihonbuzz のビジュアルアイデンティティにおいて最も重要な要素です。Nihonbuzz には 2 つのブランドがあります：**Nihonbuzz**（企業として）と **Nihonbuzz Academy**（教育部門）。

## Nihonbuzz Academy ロゴ

アカデミーロゴは、コース、教材、学習イベントなど教育関連のコンテンツで使用します。

## 使用ルール（基本）

- ロゴの比率を変更しないでください。
- 公式カラー以外に変更しないでください。
- コントラストが弱い背景には置かないでください。
